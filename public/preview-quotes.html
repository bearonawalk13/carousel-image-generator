API Usage

// Endpoint (independent from carousel generator)
POST https://carousel-image-generator.vercel.app/api/quote

// Request Body
{
  "theme": "green",     // "green" | "white" | "black"
  "platform": "instagram", // "instagram" | "linkedin" | "facebook" | "threads"
  "data": {
    "text": "Your quote goes here"
  }
}

// Response: PNG image

// Example with curl:
curl -X POST https://carousel-image-generator.vercel.app/api/quote \
  -H "Content-Type: application/json" \
  -d '{"theme":"green","platform":"instagram","data":{"text":"Hypervigilance is not paranoia. It is memory."}}' \
  --output quote.png
